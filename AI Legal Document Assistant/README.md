# 🧠 AI Legal Document Assistant

A simple, intelligent web application designed to help analyze legal tribunal documents (such as CESTAT orders) using Python and NLP techniques.

---

## 🚀 Features

* 📄 Upload and extract text from PDF documents
* 🧠 Structured legal summary generation
* 🔍 Keyword-based search within documents
* 🌐 Hindi translation support
* 🤖 AI-powered Q&A (using local NLP models)
* 🎯 Custom view modes (Decision / Financial / Full Summary)

---

## 🛠️ Tech Stack

* Python
* Streamlit
* pdfplumber
* Transformers (HuggingFace)
* Argos Translate

---

## 📌 Use Case

Best suited for:

* Tribunal/legal documents with structured text
* CESTAT-style orders

---

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

* Built a hybrid system combining rule-based NLP and local AI models
